CSS

Don't name classes on what they look like. Always use classes, avoid styling tags directly.

For the sake of simplicity, if you intend to style everything on the same page, use the HTML tags. Otherwise use classes.

You can merge classes by permutating them with class names.

Classes overpower tag selectors.

CSS Layouts

Floats are outdated, only use grids and flexbox.

Always set flex on the parent, so that you can layout the children.
